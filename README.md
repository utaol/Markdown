# 查看器说明

这个版本是纯 `index.html` 页面版本

## 现在能做什么

- 打开首页后直接选择文档
- 把 Markdown 渲染成阅读页面
- 把 Markdown 里面的原生 HTML 按真实界面显示出来
- 支持在文档里点击其他 `.md` 链接继续阅读

## 当前实现方式

因为浏览器直接双击打开本地 `index.html` 时，不能安全地自动扫描 `./md` 文件夹，所以当前版本把可查看文档内置在了 `index.html` 里。

## 如果以后要新增文档

1. 把新的 `.md` 文件放到 `md` 文件夹
2. 打开 `index.html`
3. 在 `BUILTIN_DOCS` 数组里新增一个对象，填入：

```js
{
  name: "你的文件名.md",
  path: "你的文件名.md",
  content: "这里放 Markdown 文本内容"
}
```

## 备注

如果你后面想要“真正自动扫描 `./md` 下所有文件”，那就不能继续只靠双击本地 `index.html` 了，需要改成本地服务版。当前这个版本的重点是：不需要额外脚本，打开就能用。
