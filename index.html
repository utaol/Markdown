Markdown Viewer

首页选文档，阅读页把 Markdown 里的 HTML 正常显示出来

双击 index.html 后会直接进入首页，选择文档后进入阅读页， 页面会把 Markdown 渲染出来，同时保留其中原生 HTML 的真实界面效果。

双击 index.html 即可打开 首页可直接选文档 支持内嵌 HTML 支持文档内继续跳转 .md
